PCA + DBSCAN Clustering App

This project demonstrates Unsupervised Learning using:

Principal Component Analysis (PCA)

DBSCAN Clustering

Streamlit for interactive visualization

Features

Upload CSV dataset

Automatic numeric feature selection

Dimensionality reduction using PCA

Density-based clustering using DBSCAN

Interactive parameter tuning

Visual cluster representation

Technologies Used

Python

Scikit-learn

Streamlit

Matplotlib

Pandas

How to Run
pip install streamlit scikit-learn matplotlib pandas
streamlit run app.py
